HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Elements that are nested inside other elements are called child elements. The element that holds child elements is the parent element.
  • Src: defines the location or source of image file.
  • Alt text: describes the appearance and functionality of the image.
  • The viewport meta tag controls the page's dimensions and scaling with various devices.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Inline CSS is used to style an element directly in the HTML file.
  • The internal stylesheet is used to embed CSS rules directly in the HTML file using the "style" element.
  • The external stylesheet is used to link an external file using the link element in the "head" section of HTML.
  • By using a comma, two or more elements that share the same styling can be grouped together to allow for fewer lines of code.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • Pull request: merges feature branch into main/base branch.
  • Github issues: task management tool for keeping track of progress.
  • Git commits are a description of changes made in the code.
  • We create a new branch whenever we add new features to the app.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Using conditional statements allows us to interrupt control flow.
  • Developers use console.log to debug and test their code.
  • Arrays store multiple items with different data types in a single variable.
  • An index is a unique number used to access individual data inside an array.
